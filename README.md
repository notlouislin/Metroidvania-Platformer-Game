## About

**Metroidvania-Platformer-Game** is a 2D singleplayer platformer inspired by classic metroidvania games. Built using the [Godot Engine](https://godotengine.org/) and scripted entirely in [GDScript](https://docs.godotengine.org/en/stable/tutorials/scripting/gdscript/index.html), the game includes platforming mechanics, animated characters, AI enemies, and various gameplay elements.

## Features

- Responsive 2D movement and jumping
- Enemies with simple AI behaviors
- Environmental interactions (platforms, hazards, doors)
- Animated character sprites
- Collectibles and pickups
- Level progression (expandable)
- Fully open-source under the MIT License

## Getting Started

### Prerequisites

- [Godot Engine 3.x or 4.x](https://godotengine.org/download)
- Git (optional, for cloning)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/notlouislin/Metroidvania-Platformer-Game.git
   ```
2. **Open the project:**
   - Launch Godot.
   - Click Import and select the `project.godot` file in the cloned directory.

3. **Play the Game:**
   - Press F5 or click Play Scene to start.

## Controls

| Action           | Key         |
|------------------|------------|
| Move Left/Right  | Arrow keys or A/D |
| Jump             | Spacebar   |
| Attack           | Z          |
